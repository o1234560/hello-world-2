# CSS 网络安全字体

下面列出了适用于 HTM L和 CSS 的最佳 Web 安全字体：

- Arial (sans-serif)
- Verdana (sans-serif)
- Helvetica (sans-serif)
- Tahoma (sans-serif)
- Trebuchet MS (sans-serif)
- Times New Roman (serif)
- Georgia (serif)
- Garamond (serif)
- Courier New (monospace)
- Brush Script MT (cursive)

【举例】

```css
p{font-family:"Times New Roman", Times, serif}
```
