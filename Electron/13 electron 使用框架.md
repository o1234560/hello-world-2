# electron 使用框架

## 1.在项目中自己进行配置

在`Vue`或`React`项目中添加`electron`（需要自己进行配置）。

## 2.使用electron-vite创建项目（无需自己配置）

直接使用[electron-vite](https://cn.electron-vite.org/)创建`electron+Vue`或`electron+React`项目（推荐）。

**方式一：使用最新版本**

```shell
npm create @quick-start/electron@latest
```

**方式二：使用稳定版本**

```shell
npm create @quick-start/electron
```

然后根据提示操作即可。
