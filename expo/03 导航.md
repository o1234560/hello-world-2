# 在 Expo 和 React Native 应用中的导航

了解在 Expo 和 React Native 项目中集成导航的推荐方法。

---

核心的 React Native 库不包含内置的导航解决方案，因此你可以选择最适合你需求的导航库。对于 Expo 和 React Native 应用，通常的选择是 [React Navigation](https://react-navigation.nodejs.cn/) 或 [Expo Router](https://expo.nodejs.cn/router/introduction/)。

## [为什么 React Native 应用需要导航库](https://expo.nodejs.cn/develop/app-navigation/#%E4%B8%BA%E4%BB%80%E4%B9%88-react-native-%E5%BA%94%E7%94%A8%E9%9C%80%E8%A6%81%E5%AF%BC%E8%88%AA%E5%BA%93)

React Native 核心包括基本的 UI 组件、触摸处理、设备 API 和网络功能，但不包括存储、相机、地图、大多数设备传感器以及 导航 等功能！这些功能由社区库提供支持。

## [React Navigation](https://expo.nodejs.cn/develop/app-navigation/#react-%E5%AF%BC%E8%88%AA)

React Navigation 是一个基于组件的导航库，在整个 React Native 生态系统中被广泛使用。它允许你完全通过代码组合堆栈、标签和抽屉导航器，从而实现复杂的流程、自定义过渡效果以及应用特定的用户体验模式。

该库提供针对特定平台的界面风格，带有流畅的动画和手势，统一的移动端和网页路由，自动深度链接、静态配置的类型路由，并且高度可定制。

## [Expo Router（建议用于 Expo 项目）](https://expo.nodejs.cn/develop/app-navigation/#expo-%E8%B7%AF%E7%94%B1%E5%BB%BA%E8%AE%AE%E7%94%A8%E4%BA%8E-expo-%E9%A1%B9%E7%9B%AE)

Expo Router 是一个基于文件的路由库，适用于 Expo 和 React Native 项目，并且构建在 React Navigation 之上。通过遵循 app 目录约定，它将文件转换为路由，并与 Expo 集成，用于 [Expo CLI](https://expo.nodejs.cn/more/expo-cli/) 和打包，无需额外设置。该库还增加了诸如类型化路由、动态路由、开发中的延迟打包、网页静态渲染以及自动深度链接等功能。

使用 `npx create-expo-app@latest --template default@sdk-55` 创建的新 Expo 项目默认包含 Expo Router，因此你可以快速实现跨平台导航，同时在需要时仍然可以使用 React Navigation API。
