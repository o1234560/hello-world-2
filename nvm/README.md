# nvm

nvm 是 node 版本控制器。

最好先安装 nvm，再安装 node 。

## 显示可用的 node

```powershell
nvm ls available
```

## 显示已经安装的 node

```powershell
nvm ls
```

## 安装指定版本的 node

```powershell
nvm install node版本号
```

## 卸载指定版本的 node

```powershell
nvm uninstall node的版本号
```

## 使用指定版本的 node

```powershell
nvm use node的版本号
```
